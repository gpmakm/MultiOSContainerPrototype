## Multi OS Container
```
It is a protype model using JAVA. Actually there is an aim to prove that using same resources of a computer it is possible to switch to three Operating systems.
Hereby, i have used JAVA because it is a very good Object-Oriented language which is very easy to learn. 
The reason to choose the language is that i can create multiple classes representing OSs. 
Collaborators are very welcome to improve the project.
On the date 13/04/2026, the container is running well. I have tried it using a loop. I am running the loop using dynamic values then it is working fine for 2 OSs. Similarly it may work for three also.
It is saving context in the memory variable, and for the rest you can see the code.

```
| Variable types | VariableNames |
|----------------|---------------|
| Global Variables | memoryUsage, MAX_MEMORY |
|Local Variables | status, working, osname |




| Class 1 | Class  2 | Class 3 |
|----------|----------|----------|
| Name of 1st OS    | Name of 2nd OS   | Name of 3rd OS   |
| A function to run(loop)   | A function  | A function |
| Working status  | Working status | Working status |
