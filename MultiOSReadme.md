## Multi OS Container
```
It is a protype model using JAVA. Actually there is an aim to prove that using same resources of a computer it is possible to switch to three Operating systems.
Hereby, i have used JAVA because it is a very good Object-Oriented language which is very easy to learn. 
The reason to choose the language is that i can create multiple classes representing OSs. 
Collaborators are very welcome to improve the project.
```
